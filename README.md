# Student Management System

This project is a simple student management system implemented in C++. It allows users to perform various operations such as inserting, searching, updating, and deleting student records.

## Features

- Insert student records with ID, name, last name, qualification, and phone number.
- Search for student records by ID.
- Update existing student records.
- Delete student records by ID.
- Display all student records.
- Display student records from a file.

## Prerequisites

Before running the program, ensure you have the following installed:

- C++ compiler
- Basic understanding of C++ programming

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/your-username/student-management-system.git
```

2. Compile the program:

```bash
g++ main.cpp -o student_management_system
```

3. Run the program:

```bash
./student_management_system
```

4. Follow the on-screen instructions to perform various operations on student records.

## Usage

- Choose an operation from the menu by entering the corresponding number.
- Follow the prompts to input data or perform actions.
- View the output displayed on the screen.
